# 📊 Curso: Dados Abertos — Como Usar Portais de Transparência

> **Nível:** Intermediário
> **Duração estimada:** 5–6 horas
> **Categoria:** Ferramentas e Dados

## 1. Visão Geral

O Brasil tem uma das legislações mais avançadas do mundo em **transparência pública**: a **Lei de Acesso à Informação (LAI — Lei 12.527/2011)** garante que qualquer cidadão pode pedir dados ao governo. Mas os dados só viram informação útil quando alguém sabe **acessar, cruzar e interpretar**. Este curso ensina como fazer isso na prática.

## 2. Público-Alvo

- Cidadãos que querem ir além da leitura de reportagens
- Jornalistas, pesquisadores, cientistas de dados
- Professores e estudantes
- Qualquer pessoa interessada em controle social

## 3. Conteúdo Programático

### Módulo 1 — Fundamentos
- 📜 **LAI (Lei 12.527/2011)** — direito de acesso a informações públicas
- 📜 **LGPD (Lei 13.709/2018)** — proteção de dados pessoais
- 📜 **Lei 14.129/2021** — Governo Digital
- 📜 **Decreto 8.777/2016** — Política de Dados Abertos
- 📊 **5 estrelas de dados abertos** (Tim Berners-Lee)

### Módulo 2 — Onde Encontrar Dados
- 🌐 **Portal Brasileiro de Dados Abertos** — [dados.gov.br](https://dados.gov.br/)
- 💵 **Portal da Transparência** — [portaldatransparencia.gov.br](https://portaldatransparencia.gov.br/)
- 🏛️ **Portal do TCU** — gastos e auditorias
- 🧮 **Tesouro Nacional** — execução orçamentária
- 🏥 **DATASUS** — saúde
- 🎓 **INEP / QEdu** — educação
- 🛡️ **Atlas da Violência / IPEA** — segurança e indicadores sociais
- 💧 **SNIS** — saneamento

### Módulo 3 — Formatos de Dados
- 📋 **CSV** — tabelas em texto simples
- 📊 **XLSX** — planilhas Excel
- 🗂️ **JSON** — dados estruturados para APIs
- 🌐 **HTML** — páginas da web (raspagem com cuidado!)
- 🗺️ **KML/GeoJSON** — dados geográficos
- 📚 **PDF** — documentos (pode exigir OCR)

### Módulo 4 — Ferramentas Básicas
- 📊 **Google Sheets / Excel** — filtros, tabelas dinâmicas, gráficos
- 📈 **Datawrapper / Flourish** — visualização de dados sem código
- 🐍 **Python + Pandas** — análise de dados
- 🗺️ **Google My Maps / QGIS** — mapas interativos
- 🔍 **Google Operadores** — buscas avançadas

### Módulo 5 — Como Pedir um Dado (e-sic / Fala.BR)
1. Acesse o [Fala.BR](https://falabr.cgu.gov.br/) (antigo e-SIC)
2. Faça o login com conta gov.br
3. Escreva o pedido de forma **objetiva** e **específica**
4. Acompanhe o prazo de resposta (20 dias prorrogáveis por mais 10)
5. Se negado, recorra à **CMRI** ou **autoridade superior**

## 5. Prática / Atividade

**Atividade: Cruzar 3 fontes de dados**
1. Escolha sua cidade e identifique 3 indicadores (ex.: repasses da saúde, IDEB das escolas, ocorrências criminais)
2. Baixe os dados de **3 fontes oficiais diferentes**
3. Use Google Sheets para criar uma planilha única consolidada
4. Gere **1 gráfico** que mostre a correlação (ou não) entre os indicadores
5. Publique sua análise em uma issue do GitHub com a hashtag `#cursodadosabertos`

**Desafio: Fazer um pedido de LAI**
1. Acesse o [Fala.BR](https://falabr.cgu.gov.br/)
2. Faça um pedido de informação real ao seu município
3. Documente a resposta (e a demora) numa issue do projeto
