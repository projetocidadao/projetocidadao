# 🎓 Cursos

Módulo educacional do Projeto Cidadão — cursos colaborativos sobre transparência pública, fiscalização e direitos do cidadão.

> Antes de criar um curso, leia [CURSOS_GOVERNANCA.md](../CURSOS_GOVERNANCA.md) — há um período de incubação, a regra das 3 aprovações e o sistema de pontos pioneiros.

## Índice de Cursos

### 1. Fundamentos da Fiscalização
- 🏛️ [Órgãos Públicos](./orgaos-publicos.md) — Estrutura dos 3 poderes e como eles se relacionam
- 📜 [Direito Constitucional](./direito-constitucional.md) — Direitos do cidadão previstos na CF/88
- 🛡️ [Fiscalização Cidadã](./fiscalizacao-cidada.md) — Práticas de controle social na sua cidade

### 2. Ferramentas e Dados
- 🗂️ [Licitações e Contratos](./licitacoes-contratos.md) — Como ler e fiscalizar editais
- 📊 [Dados Abertos](./dados-abertos.md) — Como usar portais de transparência
- 💻 [Tecnologia e Transparência](./tecnologia-transparencia.md) — Ferramentas digitais para o controle social

## Como cada curso é estruturado

Cada curso segue o mesmo template, para facilitar a consulta e a curadoria:

1. **Visão Geral** — o que você vai aprender
2. **Público-Alvo** — para quem é o curso
3. **Conteúdo Programático** — módulos e tópicos
4. **Fontes de Estudo** — onde aprofundar
5. **Prática / Atividade** — como aplicar imediatamente
6. **Avaliação** — como medir seu progresso

## Como contribuir com um curso novo

1. **Passe o período de incubação** (análise automática de IA)
2. **Crie o curso** seguindo o template
3. **Submeta para aprovação** — espere 3 aprovações de usuários avançados
4. **Acompanhe** sugestões e melhorias da comunidade

📌 *Cursos são bens comuns — conhecimento sobre o Estado é direito de todos.*
