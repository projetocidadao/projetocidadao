# 💻 Tecnologia e Transparência

> *"A transparência é o antídoto contra a corrupção."*
> — Projeto Cidadão

## 1. Visão Geral

Este curso ensina como a tecnologia pode ser usada para promover a transparência pública e como o cidadão pode usar ferramentas digitais para fiscalizar o poder público.

## 2. Público-Alvo

- Cidadãos interessados em tecnologia e transparência
- Desenvolvedores e programadores
- Jornalistas de dados
- Estudantes de ciência da computação, direito e administração pública
- Servidores públicos de TI

## 3. Conteúdo Programático

### Módulo 1 — Dados Abertos
- **O que são** dados abertos
- **Formatos** — CSV, JSON, XML, RDF
- **Licenças** — Creative Commons, Open Data Commons
- **Catálogo** — dados.gov.br

### Módulo 2 — Ferramentas de Análise de Dados
- **Python** — pandas, matplotlib, seaborn
- **R** — ggplot2, dplyr
- **SQL** — consultas em bancos de dados públicos
- **Excel/Google Sheets** — análise básica

### Módulo 3 — Visualização de Dados
- **Gráficos** — barras, linhas, pizza, mapas
- **Dashboards** — Tableau, Power BI, Looker Studio
- **Mapas** — Google Maps, Leaflet, Mapbox
- **Infográficos** — Canva, Adobe Illustrator

### Módulo 4 — Web Scraping Ético
- **O que é** web scraping
- **robots.txt** — respeitar as regras do site
- **Rate limiting** — não sobrecarregar o servidor
- **Termos de uso** — respeitar as condições

### Módulo 5 — APIs Públicas
- **Portal da Transparência** — API de gastos, programas, servidores
- **Câmara dos Deputados** — API de proposições, deputados
- **Senado Federal** — API de projetos, senadores
- **CNJ** — API de processos judiciais
- **IBGE** — API de dados estatísticos

### Módulo 6 — Segurança e Privacidade
- **LGPD** — Lei Geral de Proteção de Dados
- **Anonimização** — como proteger a identidade
- **Criptografia** — como proteger dados sensíveis
- **Boas práticas** — segurança digital

## 5. Prática / Atividade

**Atividade:** Escolha um tema público (ex.: gastos com saúde, licitações, votações) e:
1. Busque a API pública correspondente
2. Faça uma consulta simples (ex.: gastos por município)
3. Visualize os dados em um gráfico
4. Compartilhe sua análise na comunidade
