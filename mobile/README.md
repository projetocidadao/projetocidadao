# Mobile - React Native

App móvel do Projeto Cidadão para iOS e Android.

## Instalação

```bash
cd mobile
npm install
```

## Executar

```bash
# Android
npx react-native run-android

# iOS
npx react-native run-ios
```

## Estrutura

```
src/
  screens/      # Telas do app
  components/  # Componentes reutilizáveis
  services/    # Chamadas API
  navigation/  # Navegação
```

## API

Conecta com a API Python em `localhost:8000`.
