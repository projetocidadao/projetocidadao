# Projeto Cidadão

Sistema brasileiro de transparência pública que mapeia os 3 poderes (Executivo, Legislativo, Judiciário).

## Fontes de Dados

Este projeto integra dados das seguintes fontes públicas:

- **Portal da Transparência** - Repasses, programas sociais
- **Compras Governamentais** - Licitações, contratos
- **Câmara dos Deputados** - Proposições, votações
- **Senado Federal** - Projetos de lei, senators
- **CNJ/PJe** - Processos judiciais

## Estrutura

```
/backend          # API Python (FastAPI)
/mobile          # App React Native
/integracoes/    # Módulos de integração
```
